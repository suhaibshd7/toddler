# 🎹 Toddler Keyboard

A colourful, fullscreen musical keyboard toy for toddlers. Tap anything — every press shows a big emoji, fires a burst of colour, and plays a sound.

## Features

- 🎹 6 instruments: Piano, Retro, Bell, Bubble, Xylophone, Guitar
- 🌈 6 musical scales (all sound great — no wrong notes)
- 🐶 Animal synth sounds + .ogg audio files if present
- 🍎 Fruit pops, letter jingles, vehicle & world emoji
- ⚙️ Content manager — toggle/weight each emoji group
- 🔒 Kid-lock: hold ✕ 2 seconds to exit
- 📱 Works on Android, Windows, and any modern browser
- 💾 Sounds cached in IndexedDB after first load
